    • __construct() — Create a new TwilioCapability with zero permissions. Next steps are to grant access to resources by configuring this token through the functions allowXXXX.
    • addClaim() — Allows to set custom claims, which then will be encoded into JWT payload.
    • allowClientIncoming() — If the user of this token should be allowed to accept incoming connections then configure the TwilioCapability through this method and specify the client name.
    • allowClientOutgoing() — Allow the user of this token to make outgoing connections.
    • allowEventStream() — Allow the user of this token to access their event stream.
    • generateToken() — Generates a new token based on the credentials and permissions that previously has been granted to this token.
