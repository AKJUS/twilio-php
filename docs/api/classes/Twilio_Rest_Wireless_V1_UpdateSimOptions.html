  • setCallbackMethod() — The HTTP method we should use to call `callback_url`. Can be: `POST` or `GET`. The default is `POST`.

When the SIM transitions from `new` to `ready` or from any status to `deactivated`, we call this URL when the status changes to an intermediate status (`ready` or `deactivated`) and again when the status changes to its final status (`active` or `canceled`).
